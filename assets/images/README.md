# MycoDx 직접 제작 약도

## 포함 파일
- mycodx-map-guide.svg
- html-snippet.html
- css-snippet.css

## 적용 방법
1. `mycodx-map-guide.svg`를 저장소의 `assets/images/` 폴더에 넣습니다.
2. `index.html`의 기존 `mapCard` 영역을 `html-snippet.html` 내용으로 교체합니다.
3. `css-snippet.css` 내용을 `css/style.css` 맨 아래 또는 `background-override.css` 맨 아래에 추가합니다.
